# Plano 04-01: Camada de Persistência Local Robusta

## 🎯 Objetivo
Implementar uma infraestrutura de dados que permita ao Life OS funcionar offline, garantindo que o estado do servidor (Supabase) seja persistido localmente e que as mutações feitas offline sejam enfileiradas e sincronizadas automaticamente ao recuperar a conexão.

## 🧱 Must-Haves
- Persistência do `QueryClient` (TanStack Query) usando `idb-keyval`.
- Configuração de `mutationDefaults` para persistência de ações offline (Habits, Tasks, Transactions).
- Integração com `@capacitor-community/sqlite` para armazenamento nativo robusto.
- Monitoramento de status de conexão (`online`/`offline`) integrado à UI.

## 📋 Tarefas

<tasks>
- [x] **Instalação de Dependências**: Adicionar `idb-keyval`, `@capacitor-community/sqlite` e plugins de persistência do TanStack.
- [x] **Configuração do Persister**: Implementar o `createSyncStoragePersister` para salvar o cache do TanStack Query no IndexedDB.
- [x] **Mutation Queue Setup**: Configurar `setMutationDefaults` no `QueryClient` para os principais domínios (Habits, Tasks).
