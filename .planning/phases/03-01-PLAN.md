# Plano 03-01: Universal Scaffolding (Electron + Capacitor)

## 🎯 Objetivo
Configurar a fundação multiplataforma do Life OS, permitindo que o mesmo código-fonte React seja empacotado para Desktop (via Electron) e Android (via Capacitor).

## 🧱 Must-Haves
- Estrutura do Electron instalada e funcional (`electron`, `vite-plugin-electron`).
- Estrutura do Capacitor instalada e inicializada (`@capacitor/core`, `@capacitor/android`).
- Configuração de build unificada: `npm run build` generates the web artifact that both wrappers consume.
- Ajuste de caminhos (Assets) para funcionar tanto em `file://` (Electron) quanto em `capacitor://` (Android).

## 📋 Tarefas

<tasks>
- [x] **Instalação Multi-Runtime**: 
    - Adicionar dependências Electron: `electron`, `vite-plugin-electron`, `electron-builder`.
    - Adicionar dependências Capacitor: `@capacitor/core`, `@capacitor/cli`, `@capacitor/android`.
- [x] **Inicialização Android**: 
    - Executar `npx cap init` e adicionar a plataforma android (`npx cap add android`).
- [x] **Scaffolding Electron**: 
    - Criar `electron/main.ts` e `electron/preload.ts` configurados para carregar o build do Vite.
- [x] **Scripts Universais**: 
    - Criar `android:dev`, `android:build`, `electron:dev` e `electron:build` no `package.json`.
- [x] **Configuração Vite**: 
    - Garantir que `base: './'` esteja configurado para suportar protocolos de arquivos nativos.
</tasks>

## 🧪 Verificação (DoD)
- [x] `npm run electron:dev` abre a janela desktop.
- [x] `npx cap open android` abre o Android Studio com o projeto pronto para rodar.
- [x] Um único componente (ex: Dashboard) é alterado e a mudança é refletida em ambas as plataformas após o build.
