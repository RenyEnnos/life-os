# Plan: 08-04 Achievement Gallery

**Phase:** 08 - Gamification Engine
**Plan:** 08-04
**Status:** Ready to execute

## Goal
Provide a central place for users to view all available achievements and their current collection status.

## Context
- The system already has a catalog of achievements in the backend.
- Users need to see what they've earned and what's left to unlock to stay motivated.

## Task Breakdown
1. **Rewards Page/Tab**: Create or update the Rewards page to include a comprehensive achievement gallery.
   - Files: `src/features/rewards/pages/RewardsPage.tsx`
2. **Achievement Grid**: Implement a grid view showing "locked" (greyed out) and "unlocked" (colored) achievement cards.
   - Files: `src/features/rewards/components/AchievementGrid.tsx`
3. **Detail View**: Show achievement description and date unlocked when clicked.

## Verification
- [ ] All achievements from the database catalog are listed.
- [ ] Unlocked achievements are visually distinct.
- [ ] User can see exactly how many XP they earned for each badge.
