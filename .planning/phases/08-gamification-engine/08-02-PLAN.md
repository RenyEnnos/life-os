# Plan: 08-02 Automatic XP Integration

**Phase:** 08 - Gamification Engine
**Plan:** 08-02
**Status:** Ready to execute

## Goal
Ensure that completing tasks and habits automatically awards XP to the user, providing immediate feedback and reinforcement.

## Context
- Users need a tangible reward for their productivity to maintain engagement.
- We need to coordinate between Task/Habit features and the Rewards feature.

## Task Breakdown
1. **XP constants**: Define standard XP rewards (e.g., Task: 100XP, Habit: 50XP).
2. **Task Integration**: Update `useTasks` hook to call `rewardsApi.addXp` when a task is marked as 'done'.
   - Files: `src/features/tasks/hooks/useTasks.ts`
3. **Habit Integration**: Update `useHabits` hook to award XP when a habit is logged.
   - Files: `src/features/habits/hooks/useHabits.ts`
4. **Optimistic XP**: Optionally update the local `life-score` query data optimistically to show immediate XP gain.

## Verification
- [x] Marking a task as 'done' increments the user's XP in the XP Bar.
- [x] Logging a habit increments XP.
- [x] XP Gain is reflected without a full page reload.
