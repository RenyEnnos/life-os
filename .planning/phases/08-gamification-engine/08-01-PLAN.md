# Plan: 08-01 XP Bar & UI Foundation

**Phase:** 08 - Gamification Engine
**Plan:** 08-01
**Status:** Ready to execute

## Goal
Establish the visual foundation for the gamification system by updating types and adding a global XP progress bar.

## Context
- The user currently has no way to see their level or XP progress in the UI.
- The backend already provides these values through the `/api/rewards/score` endpoint.

## Task Breakdown
1. **Type Update**: Update `LifeScore` in `shared/types.ts` to include `current_xp`, `level`, and `xp_to_next_level`.
   - Files: `shared/types.ts`
2. **UserLevelStatus Component**: Create a premium XP bar component that shows the current level, XP progress, and a circular progress indicator or horizontal bar.
   - Files: `src/features/user/components/UserLevelStatus.tsx`
3. **Sidebar/Header Integration**: Add the `UserLevelStatus` component to the top of the Sidebar or Header.
   - Files: `src/app/layout/Sidebar.tsx`

## Verification
