# Plan: 04-03 Kanban Interaction & DND

**Phase:** 04 - Task Management (Kanban)
**Plan:** 04-03
**Status:** Ready to execute

## Goal
Implement drag-and-drop functionality for moving tasks between status columns with smooth animations and optimistic updates.

## Context
- The user expects an interactive Kanban board where they can easily reorder and move tasks.
- Updates to task status during drag-and-drop must be reflected immediately in the UI (optimistic updates).

## Task Breakdown
1. **DND Integration**: Install and configure `@dnd-kit/core` and `@dnd-kit/sortable` for the Kanban columns and cards.
   - Files: `package.json`, `src/features/tasks/components/KanbanBoard.tsx`
2. **Sortable Task Cards**: Wrap task items in a sortable context to allow them to be dragged.
   - Files: `src/features/tasks/components/SortableTaskItem.tsx`
3. **Optimistic Updates**: Use React Query (already in the project) to perform optimistic UI updates when a task is moved.
   - Files: `src/features/tasks/hooks/useTasks.ts`
4. **Drag Overlay & Animations**: Add a drag overlay for a premium look during the drag action.
   - Files: `src/features/tasks/components/KanbanBoard.tsx`

## Verification
- [ ] Task is moved between columns and its status updates in the backend.
- [ ] UI reflects the move immediately with smooth animations.
- [ ] In case of API failure, the task reverts to its original column.

## Agent Assignments
- **Interactions**: `frontend-specialist`
- **Animations**: `framer-motion` (implemented by agent)
