# Tech Stack

## Frontend
- **Framework:** React 18
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS, Framer Motion (Animations), Radix UI (Primitives)
- **State Management:** Zustand, React Query (@tanstack/react-query), React Context
- **Routing:** React Router DOM (v7)

## Backend / API
- **Server:** Node.js, Express
- **Database:** PostgreSQL (pg), Supabase (Auth, Database)
- **Caching/State:** node-cache
- **ORM / Query Builder:** Supabase JS Client

## Testing
- **Unit/Integration:** Vitest, React Testing Library
- **E2E:** Playwright
- **Linting/Formatting:** ESLint, Prettier

## Others
- **PWA:** vite-plugin-pwa, workbox
- **Analytics:** Custom AnalyticsProvider
- **Icons:** Lucide React, Remix Icon
